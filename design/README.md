# 🎨 Arithmetick 

This project is a simple app that performs conversions between some basic measurement units. This design is meant to be a learning project and can be used as a starting point for other projects.

# 🎚️ Layers

Here's a brief high-level overview of the layers in this design: 

- `Wireframe` - Used to setup the basic idea of how the app should work
- `iOS UI` - Dark and Light sample layouts with just a simple setup
- `Android UI` - Material themed UI
- `Web UI` - Basic idea using Bootstrap elements 
- `Colors` - Stores some of the colors used in the other layers
