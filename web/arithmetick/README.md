# 🕸️ Arithmetick

This project is a simple app that performs conversions between some basic measurement units. This app is meant to be a learning app and can be used as a starting point for other projects.

# 📦 Packages 

The following additional packages were used in this app: 

- [DaisyUI](https://daisyui.com) - An extension for TailwindCSS that adds more components to the library
- [Page.js](https://visionmedia.github.io/page.js/) - A small routing library 
- [UOM](https://github.com/dividab/uom) - Contains conversions for most units of measurements 
- [reliable-round](https://github.com/codeandcats/reliable-round) - Library for providing more accurate rounding methods
- [Vitest](https://vitest.dev) - Testing library built by the same team behind Vite
